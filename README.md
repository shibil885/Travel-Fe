# 🌍 Travel Package Management System – Frontend

A modern travel booking web application built with Angular. Users can explore travel packages, agencies can manage their listings, and admins can oversee the entire platform. Designed to be responsive, modular, and scalable.

## 🚀 Tech Stack
- **Framework**: Angular 17 (Standalone)
- **State Management**: NgRx
- **UI**: Tailwind CSS, Angular Material
- **Routing**: Angular Router
- **HTTP**: HttpInterceptorFn, Cookie-based Auth
- **Other**: Responsive design, Lazy loading, SSR-ready

## 🎯 Features
- JWT-based authentication via cookies
- Role-based access (user, agency, admin)
- Real-time UI updates using NgRx
- Booking, wallet payments, cancellation policy, and refund system
- Notification system for all user types
- Responsive design with Tailwind CSS
- Chat system integration (via WebSocket)
- Dynamic search, sorting, filtering, and pagination
